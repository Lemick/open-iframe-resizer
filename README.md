# Open Iframe Resizer

## Overview

`iframe-resizer` is a modern lightweight for resizing iframes dynamically.
It is shipped under an MIT license, thus usable in commercial projects.

## Getting started

### Browser
#### ES6 modules

```html
<script type="module">
  import { initialize } from "/dist/index.js";

  initialize({}, "#iframe-id");
</script>
```
